Laplace Transform

An operator takes a function as input and produces another function. A transform does the same thing, but the output function has a different independent variable.

A Laplace operation transforms a function of a real variable t into a function of a different, complex variable s. If t is in the time domain, then s is in the frequency domain. The function in the time domain is usually denoted with a lowercase letter f(t), the function in the frequency domain is denoted with an uppercase letter F(s).

Differentiation and integration in the time domain are are transformed into algebraic division and multiplication, which simplifies the solution of differential equations to solving an algebraic equation.

Definition

The Laplace transform is defined as

F(s) = ∫0∞ f(t) e−stdt

where s=δ+iω with δ,ω∈ℝ and i=−1∈ℂ.

The inverse Laplace transform is defined as

f(t)= L−1{F}(t)= 1 2πi lim T→∞ ∫γ−iT γ+iT e st F(s)ds

where γ∈ℝ so that the contour path of integration is in the region of convergens of F(s).

In practise, it is typically more convenient to decompose a Laplace transform into know transforms of functions obtained from a table and construct the inverse by inspection. Some standard Laplace transform tables can be found at

Example: Sine and Cosine Transforms

The Laplace transform of cosat is

ℒ{cosat}(s) = ∫0∞ cos(at) e−st dt

We'll exploit the exponential form of cosine with complex exponent:

cos(at) = 12 ( eiat + e−iat )

...to write (3) as

12 ∫0∞ ( eiat + e−iat ) e−st dt = 12 ∫0∞ e −(s−ia)t dt + 12 ∫0∞ e−(s+ia)t dt

Since

∫0∞ e−λt dt = 1λ

with λ∈ℂ and ℛℯ(λ)>0, it follows that (4) can be written as

ℒ{cosat}(s) = 12 ( 1s−ia + 1s+ia )

Simplification yields the final solution:

= 12 (s+ia) + (s−ia) (s−ia) (s+ia) = 12 2s s2 − (ia)2 = s s2 + a2

The sine function can be treated similarly:

sin(at) = e iat − e −iat 2i

Example: Hooke's Law

For example, the equation of the simple harmonic oscillator (Hooke's law)

d2x(t) dt2 + kx(t) =0

To find a solution for x(t), we will transform the equation into the s domain. We start with finding the Laplace transform of the first derivative:

L{x′(t)} = sX(s) −x(0) = sX(s) −x0

Then the second derivative:

L{x″(t)} = s2X(s) − sx(0) − x′(0) = s2X(s) − sx0 − v0

Now we can apply the transformation:

L{x″(t)} + kL{x(t)} = L{0} = 0 s2X(s) − sx0 − v0 + kX(s) = 0

Isolating X(s) produces the solution in the frequency domain:

X(s) ( s2 + k ) = sx0 + v0 X(s) = sx0 + v0 s2+k = sx0 s2+k + v0 s2+k

For the inverse transformation, recall the standard Laplace pairs for k>0:

L{cos(kt)} = s s2+k L{sin(kt)} = k s2+k

The inverse Laplace transformation of (1):

x(t) = L−1{X(s)} = L−1{x0ss2+k} + L−1{v01s2+k}

Injecting (2) and (3) produces the solution in the time domain:

x(t) = x0cos(kt) + v0 k sin(kt)
